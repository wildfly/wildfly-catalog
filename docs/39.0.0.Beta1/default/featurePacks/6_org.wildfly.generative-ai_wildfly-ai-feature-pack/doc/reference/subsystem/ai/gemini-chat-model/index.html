    allowed-code-execution BOOLEAN false true Enables the model to generate and run Python code and learn iteratively from the results until it arrives at a final output.
    api-key STRING true true API key to authenticate to a Google Gemini chat model.
    civic-integrity STRING false true Defines the threshold to filter out civic integrity content.
    connect-timeout LONG false true 0 Timeout for the Google Gemini chat model.
    dangerous-content STRING false true Defines the threshold to filter out dangerous content.
    enable-enhanced-civic-answers BOOLEAN false true Enables enhanced civic answers. It may not be available for all models.
    frequency-penalty DOUBLE false true Positive values penalize tokens that repeatedly appear in the generated text, decreasing the probability of repeating content.
    harassment STRING false true Defines the threshold to filter out harassment content.
    hate-speech STRING false true Defines the threshold to filter out hate speech content.
    include-code-execution-output BOOLEAN false true Enables providing the code and the result in the output.
    include-thoughts BOOLEAN false true Enable the return of the thought summaries of the model.
    log-probs INT false true Returns the log probabilities of the top candidate tokens at each generation step. The model's chosen token might not be the same as the top candidate token at each step. Specify the number of candidates to return by using an integer value in the range of 1-20.
    log-requests BOOLEAN false true Enabling the tracing of requests to Google Gemini
    log-responses BOOLEAN false true Enabling the tracing of responses to Google Gemini
    max-output-token INT false true The number of token returned by the Google Gemini chat model.
    model-name STRING true true Name of the model served by Google Gemini.
    presence-penalty DOUBLE false true Presence penalty applied to the next token's logprobs if the token has already been seen in the response. This penalty is binary on/off and not dependant on the number of times the token is used (after the first). Use frequency-penalty for a penalty that increases with each use.
    response-format STRING false true The format of the response from Google Gemini.
    response-log-probs BOOLEAN false true If true, returns the log probabilities of the tokens that were chosen by the model at each step.
    return-thinking BOOLEAN false true Controls whether to return thinking/reasoning text (if available).
    seed INT false true When seed is fixed to a specific value, the model makes a best effort to provide the same response for repeated requests. Deterministic output isn't guaranteed.
    sexually-explicit STRING false true Defines the threshold to filter out sexually explicit content.
    stop-sequences LIST false true List of stop sequences to tell the model to stop generating content.
    streaming BOOLEAN false true false Whether to create a token streaming chat language model or not.
    temperature DOUBLE false true Temperature of the Google Gemini chat model.
    thinking-budget INT false true Parameter to guide the model on the number of thinking tokens to use when generating a response.
    top-k INT false true TopK of the Google Gemini chat model.
    top-p DOUBLE false true TopP of the Google Gemini chat model.
